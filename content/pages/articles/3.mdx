# CSS Grid Layout Mastery

CSS Grid has revolutionized web layout by providing a two-dimensional layout system that makes it easier to design complex layouts with clean, semantic HTML. This comprehensive guide will take you from Grid basics to advanced techniques.

## What is CSS Grid?

CSS Grid is a layout system that allows you to create complex layouts using rows and columns. Unlike Flexbox, which is primarily one-dimensional, Grid excels at creating two-dimensional layouts.

## Basic Grid Setup

To create a grid container, use `display: grid`:

```css
.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  grid-template-rows: 100px 200px;
  gap: 20px;
}
```

## Grid Template Columns and Rows

Define your grid structure using various units:

```css
.grid {
  /* Fixed sizes */
  grid-template-columns: 200px 300px 100px;

  /* Flexible sizes */
  grid-template-columns: 1fr 2fr 1fr;

  /* Mixed units */
  grid-template-columns: 200px 1fr 100px;

  /* Repeat function */
  grid-template-columns: repeat(3, 1fr);

  /* Auto-fit and auto-fill */
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
}
```

## Grid Areas

Use named grid areas for semantic layouts:

```css
.layout {
  display: grid;
  grid-template-areas:
    "header header header"
    "sidebar main main"
    "footer footer footer";
  grid-template-columns: 200px 1fr 1fr;
  grid-template-rows: 80px 1fr 60px;
  min-height: 100vh;
}

.header {
  grid-area: header;
}
.sidebar {
  grid-area: sidebar;
}
.main {
  grid-area: main;
}
.footer {
  grid-area: footer;
}
```

## Grid Item Placement

Place items explicitly using line numbers or names:

```css
.item {
  /* Using line numbers */
  grid-column: 1 / 3;
  grid-row: 2 / 4;

  /* Using span */
  grid-column: span 2;
  grid-row: span 3;

  /* Shorthand */
  grid-area: 2 / 1 / 4 / 3; /* row-start / col-start / row-end / col-end */
}
```

## Auto-Placement and Grid Flow

Control how items are automatically placed:

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-auto-flow: row; /* default */
  /* grid-auto-flow: column; */
  /* grid-auto-flow: row dense; */
  grid-auto-rows: minmax(100px, auto);
}
```

## Responsive Grid Layouts

Create responsive layouts without media queries:

```css
.responsive-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
  gap: 20px;
}

/* With media queries for more control */
.advanced-grid {
  display: grid;
  gap: 20px;
  grid-template-columns: 1fr;
}

@media (min-width: 768px) {
  .advanced-grid {
    grid-template-columns: repeat(2, 1fr);
  }
}

@media (min-width: 1024px) {
  .advanced-grid {
    grid-template-columns: repeat(3, 1fr);
  }
}
```

## Alignment and Justification

Control alignment within the grid:

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 200px);
  grid-template-rows: repeat(2, 150px);

  /* Align the entire grid within the container */
  justify-content: center; /* start, end, center, space-between, space-around, space-evenly */
  align-content: center;

  /* Default alignment for all items */
  justify-items: center; /* start, end, center, stretch */
  align-items: center;
}

.item {
  /* Override alignment for specific items */
  justify-self: end;
  align-self: start;
}
```

## Advanced Grid Techniques

### Overlapping Items

```css
.overlay-grid {
  display: grid;
  grid-template-columns: 1fr;
  grid-template-rows: 1fr;
}

.background,
.content {
  grid-column: 1;
  grid-row: 1;
}

.content {
  z-index: 1;
  align-self: center;
  justify-self: center;
}
```

### Subgrid (Future Feature)

```css
.parent {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
}

.child {
  display: grid;
  grid-template-columns: subgrid; /* Inherits parent's columns */
  grid-column: span 3;
}
```

## Real-World Examples

### Card Layout

```css
.card-grid {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(280px, 1fr));
  gap: 24px;
  padding: 24px;
}

.card {
  background: white;
  border-radius: 8px;
  box-shadow: 0 2px 8px rgba(0, 0, 0, 0.1);
  overflow: hidden;
  transition: transform 0.2s;
}

.card:hover {
  transform: translateY(-4px);
}
```

### Holy Grail Layout

```css
.holy-grail {
  display: grid;
  grid-template-areas:
    "header header header"
    "nav main aside"
    "footer footer footer";
  grid-template-columns: 200px 1fr 150px;
  grid-template-rows: auto 1fr auto;
  min-height: 100vh;
}

@media (max-width: 768px) {
  .holy-grail {
    grid-template-areas:
      "header"
      "nav"
      "main"
      "aside"
      "footer";
    grid-template-columns: 1fr;
  }
}
```

## Grid vs Flexbox

When to use Grid vs Flexbox:

- **Use Grid for**: Two-dimensional layouts, complex layouts, when you need precise control over both rows and columns
- **Use Flexbox for**: One-dimensional layouts, component-level layouts, when you need items to grow/shrink

## Browser Support and Fallbacks

CSS Grid has excellent browser support, but for older browsers:

```css
.container {
  /* Fallback */
  display: flex;
  flex-wrap: wrap;

  /* Grid enhancement */
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
}

.item {
  /* Fallback */
  flex: 1 1 250px;
  margin: 10px;
}

/* Feature query */
@supports (display: grid) {
  .container {
    gap: 20px;
  }

  .item {
    margin: 0;
  }
}
```
